# SkillVault dashboard - Week 10 lab

This repository contains the lab (in Amalitech GTP program) for Week 10 - SkillVault . The SKillVault is a platform that allows users to create, share, and manage their skills and knowledge. Users are also able to track their progress and see how they are doing with the tasks that are included in the selected skill.

## Features

- User authentication and authorization
- Skill creation and management
- Task creation and management
- Progress tracking
- User dashboard to view skills and tasks
- Responsive design for mobile and desktop

## Technologies Used

- Next.js
- TypeScript
- Tailwind CSS
- Prisma ORM
- PostgreSQL ( with supabase as the database provider)
- NextAuth.js for authentication
